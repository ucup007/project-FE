## Langkah-langkah Instalasi
1. Buka terminal editor dan ketikan
```js
git clone https://github.com/ucup007/project-FE
```

2. Setelah itu masuk ke direktori project dan ketikkan
```js
npm install
```

3. Jika sudah selesai terakhir tulis
```js
npm run dev
```
